This repository contains the full tutorial, code, and documentation for Support Vector Machines (SVM) applied to the Banknote Authentication dataset.

## Contents
- `notebook.ipynb` – Complete tutorial with explanations and visualisations  
- `requirements.txt` – Python dependencies  
- `README.md` – Documentation and instructions  

## Dataset
UCI Banknote Authentication dataset  
Features: variance, skewness, curtosis, entropy  
Target: 0 = genuine, 1 = forged  

## How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```

Then open `notebook.ipynb` and run each cell sequentially.
